#### 01 系统介绍

- 系统名称：基于Web的图书管理系统（manage_books）
- 开发工具：eclipse
- 开发语言：Servlet+Jsp+Java1.8
- 数据库：MySql5.7
- tomcat9.0，端口：8080

#### 02 访问系统

- 访问地址：http://localhost:8080/manage_books
- 访问用户：  系统管理员【用户名：admin 密码：admin】 普通用户【用户名：user 密码：user】

注：访问系统前提是先把系统运行起来
